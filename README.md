# 🌸 Happy Birthday Mamma 🎂💖

A beautifully crafted, heartfelt digital birthday surprise made with love, emotions, and a touch of magic ✨

This project is a **personalized interactive birthday webpage** dedicated to the most special person — *Mamma* 💕
It combines elegant design, animations, music, and memories to create an unforgettable experience.
